# Contenu de la séance 06

- Dessin avancé d'un graphe avec
  1. `nx.sprint_layout` pour calculer les positions
  1. `nx.draw_networkx_nodes` pour les sommets
  1. `nx.draw_networkx_edges` pour les arrêtes
  1. `nx.draw_networkx_labels` pour annoter les sommets
  1. `nx.draw_networkx_edge_labels` pour annoter les arrêtes
- Graphe pondéré
- `nx.shortest_path` avec des poids
- Notion de _sérialisation_
- Utilisation de `pathlib` pour interagir avec le système de fichiers
- IO vers `dot` format et `json` format depuis `nx`
